# About China Tea House

China Tea House is a free reference for Chinese tea: types, origins, caffeine, brewing, and comparisons.

## Website

<https://chinatea.house/>

## Goals

- Make Chinese tea accessible to beginners
- Provide accurate, well-structured tea data
- Offer useful tools: tea finder, brewing calculator, caffeine chart
