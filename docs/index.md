# China Tea House Docs

A public knowledge base for Chinese tea, maintained alongside the [China Tea House](https://chinatea.house/) website.

## Explore

- [Chinese Tea Dataset](./dataset) — downloadable JSON/CSV data on 100+ teas
- [Caffeine Chart](./caffeine-chart) — caffeine by tea type
- [Brewing Guide](./brewing-guide) — temperature, time, and leaf ratios
- [Tea Finder](./finder) — find a tea by mood, caffeine, or occasion
- [Tea Comparisons](./comparisons) — side-by-side comparisons
- [About](./about) — why this project exists

## Data & Assets

All datasets and charts are free to use with attribution:

- Live dataset page: <https://chinatea.house/dataset/>
- Caffeine chart: <https://chinatea.house/chinese-tea-caffeine-chart/>
- Categories chart: <https://chinatea.house/chinese-tea-categories/>
- RSS feed: <https://chinatea.house/feed.xml>

## Contribute

Found an error or want to suggest a tea? Open an issue in the [main repository](https://github.com/ArtisanStrategies/chinatea.house/issues).
