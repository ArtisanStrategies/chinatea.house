## Alert Thresholds

### Red Alerts (Immediate Action)

| Condition | Alert |
|-----------|-------|
| Organic traffic drops >30% week-over-week | Algorithm penalty or technical issue |
| Indexation rate drops below 20% | Crawl budget or quality issue |
| Core Web Vitals fail | Performance regression |
| Crawl errors spike >1% | Technical problem |

### Yellow Alerts (Investigation Needed)

| Condition | Alert |
|-----------|-------|
| Traffic growth stalls for 2+ weeks | Content or competition issue |
| Bounce rate exceeds 80% | Content quality or intent mismatch |
| Build failures >1% | Template or data issue |
| New pages not indexing within 2 weeks | Sitemap or crawl issue |

## Reporting Cadence

### Daily (Automated)
- Pages published count
- Build/deploy status
- Crawl errors (if any)

### Weekly (Manual Review)
- Traffic trends
- Indexation progress
- Top performing pages
- Keywords gaining/losing

### Monthly (Strategic Review)
- Milestone progress vs targets
- Content strategy adjustments
- Technical debt assessment
- Competitive landscape check
