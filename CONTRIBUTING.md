# Contributing to China Tea House

Thanks for your interest in improving China Tea House.

## How to contribute

- **Report errors or missing teas** by opening a GitHub issue.
- **Suggest new features** (tools, charts, guides) in [Discussions](https://github.com/ArtisanStrategies/chinatea.house/discussions).
- **Share the project** on social media, newsletters, or your own site — a link back to https://chinatea.house/ is always appreciated.
- **Use the dataset** in research, apps, or visualizations and cite the [dataset page](https://chinatea.house/dataset/) or [GitHub release](https://github.com/ArtisanStrategies/chinatea.house/releases/tag/dataset-v1.0).

## Resources

- Live site: https://chinatea.house/
- Dataset: https://chinatea.house/dataset/
- Docs site: https://artisanstrategies.github.io/chinatea.house/
- RSS feed: https://chinatea.house/feed.xml

## Code changes

This is a Python static-site generator. If you want to change code or data:

1. Open an issue describing the change.
2. Fork the repository and create a branch.
3. Make your changes and run `python -m execution.cli build --full` locally.
4. Submit a pull request.

Please keep changes focused and follow the existing style.
