Characteristics of {{ category.name_en }}

Processing

{% if category.id == 'green' %} Minimal oxidation (0-5%). Leaves are quickly heated to preserve freshness. {% elif category.id == 'white' %} Minimal processing with natural withering. Oxidation ranges from 5-15%. {% elif category.id == 'yellow' %} Similar to green tea but with a 'sealed yellowing' step creating mellower flavor. {% elif category.id == 'oolong' %} Partial oxidation (15-85%) with diverse styles from light and floral to dark and roasted. {% elif category.id == 'black' %} Fully oxidized (85-100%), resulting in bold, robust flavors. {% elif category.id == 'dark' %} Post-fermented with microbial activity, developing complex aged character. {% elif category.id == 'scented' %} Base tea layered with fresh flowers to absorb natural fragrance. {% endif %}

Typical Flavors

{% if category.id == 'green' %} Fresh, vegetal, grassy, nutty, marine, sweet {% elif category.id == 'white' %} Delicate, honey, melon, hay, subtle floral {% elif category.id == 'yellow' %} Mellow, sweet corn, chestnut, smooth {% elif category.id == 'oolong' %} Floral, fruity, creamy, roasted, mineral {% elif category.id == 'black' %} Malty, chocolate, honey, dried fruit, robust {% elif category.id == 'dark' %} Earthy, woody, leather, dates, smooth {% elif category.id == 'scented' %} Floral (jasmine, osmanthus), sweet, fragrant {% endif %}

Best Brewing

{% if category.id == 'green' %} Lower temperature (70-80°C), shorter steeps, glass or porcelain {% elif category.id == 'white' %} Moderate temperature (80-85°C), longer steeps, gaiwan or glass {% elif category.id == 'yellow' %} Similar to green (75-80°C), glass to watch leaves {% elif category.id == 'oolong' %} Higher temperature (85-95°C), multiple steeps, gaiwan or Yixing {% elif category.id == 'black' %} Near boiling (90-95°C), medium steeps, any vessel {% elif category.id == 'dark' %} Boiling water (100°C), rinse first, many steeps, Yixing ideal {% elif category.id == 'scented' %} Moderate temperature (80-85°C), short steeps to preserve aroma {% endif %}

Caffeine Level

{% if category.id == 'green' %} Moderate. L-theanine provides calm alertness. {% elif category.id == 'white' %} Low to moderate, depending on bud content. {% elif category.id == 'yellow' %} Moderate, similar to green tea. {% elif category.id == 'oolong' %} Moderate to high, varies by oxidation level. {% elif category.id == 'black' %} Moderate to high. Good morning tea. {% elif category.id == 'dark' %} Low to moderate. Fermentation reduces caffeine. {% elif category.id == 'scented' %} Varies by base tea (typically moderate for green base). {% endif %}

How to Brew {{ category.name_en }}

Gongfu Style

{% if category.id == 'green' %} Use 4-5g per 100ml, water at 75-80°C. Steep for 20-30 seconds, adding 5 seconds each steep. 4-5 steeps. {% elif category.id == 'white' %} Use 4-5g per 100ml, water at 80-85°C. Steep for 30-45 seconds initially. 5-6 steeps. {% elif category.id == 'yellow' %} Use 4g per 100ml, water at 75-80°C. Steep for 30-45 seconds. 4-5 steeps. {% elif category.id == 'oolong' %} Use 6-8g per 100ml, water at 90-95°C. Quick rinse, then 15-25 second steeps. 7-10 steeps. {% elif category.id == 'black' %} Use 5g per 100ml, water at 90-95°C. Steep for 15-20 seconds, increasing. 5-6 steeps. {% elif category.id == 'dark' %} Use 6-7g per 100ml, boiling water. Rinse twice, then 10-15 second steeps. 10-20 steeps. {% elif category.id == 'scented' %} Use 4g per 100ml, water at 80-85°C. Steep for 30 seconds. 4-5 steeps. {% endif %}

Western Style

{% if category.id == 'green' %} Use 2g per 200ml, water at 75-80°C. Steep for 2-3 minutes. 2-3 steeps. {% elif category.id == 'white' %} Use 2-3g per 200ml, water at 80-85°C. Steep for 3-4 minutes. 2-3 steeps. {% elif category.id == 'yellow' %} Use 2g per 200ml, water at 75-80°C. Steep for 2-3 minutes. 2-3 steeps. {% elif category.id == 'oolong' %} Use 3g per 200ml, water at 90°C. Steep for 2-3 minutes. 3-4 steeps. {% elif category.id == 'black' %} Use 2.5g per 200ml, water at 90-95°C. Steep for 3-4 minutes. 2-3 steeps. {% elif category.id == 'dark' %} Use 3g per 200ml, boiling water. Rinse first, steep for 3-4 minutes. 4-5 steeps. {% elif category.id == 'scented' %} Use 2g per 200ml, water at 80-85°C. Steep for 2-3 minutes. 2-3 steeps. {% endif %}
