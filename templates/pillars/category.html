Processing

{% if category.id == 'green' %} Minimal oxidation (0–5%). Leaves quickly heated after picking to preserve freshness. {% elif category.id == 'white' %} Minimal processing with gentle withering. Natural oxidation ranges 5–15%. {% elif category.id == 'yellow' %} Similar to green with unique 'sealed yellowing' step creating mellower character. {% elif category.id == 'oolong' %} Partial oxidation (15–85%) producing remarkable diversity from floral to deeply roasted. {% elif category.id == 'black' %} Full oxidation (85–100%), developing bold, robust character. {% elif category.id == 'dark' %} Post-fermented with microbial activity, transforming over months or decades. {% elif category.id == 'scented' %} Base tea layered repeatedly with fresh flowers to absorb natural fragrance. {% endif %}

Character

{% if category.id == 'green' %} Fresh, vegetal, grassy, nutty, marine, sweet {% elif category.id == 'white' %} Delicate, honey, melon, hay, subtle floral {% elif category.id == 'yellow' %} Mellow, sweet corn, chestnut, smooth {% elif category.id == 'oolong' %} Floral, fruity, creamy, roasted, mineral {% elif category.id == 'black' %} Malty, chocolate, honey, dried fruit, robust {% elif category.id == 'dark' %} Earthy, woody, leather, dates, smooth {% elif category.id == 'scented' %} Jasmine, osmanthus, rose—sweet, fragrant {% endif %}

Brewing

{% if category.id == 'green' %} 70–80°C water, shorter steeps. Glass or porcelain ideal. {% elif category.id == 'white' %} 80–85°C water, patient steeping. Gaiwan or glass. {% elif category.id == 'yellow' %} 75–80°C water. Glass shows leaf transformation. {% elif category.id == 'oolong' %} 85–95°C water, many quick steeps. Gaiwan or Yixing clay. {% elif category.id == 'black' %} 90–95°C water, medium steeps. Any vessel works well. {% elif category.id == 'dark' %} Boiling water, rinse first, many steeps. Yixing clay ideal. {% elif category.id == 'scented' %} 80–85°C water, short steeps to preserve delicate aroma. {% endif %}

Preparation

How to Brew {{ category.name_en }}

Gongfu Style

{% if category.id == 'green' %} Use 4–5g per 100ml, water at 75–80°C. Steep for 20–30 seconds, adding 5 seconds each steep. 4–5 steeps. {% elif category.id == 'white' %} Use 4–5g per 100ml, water at 80–85°C. Steep for 30–45 seconds initially. 5–6 steeps. {% elif category.id == 'yellow' %} Use 4g per 100ml, water at 75–80°C. Steep for 30–45 seconds. 4–5 steeps. {% elif category.id == 'oolong' %} Use 6–8g per 100ml, water at 90–95°C. Quick rinse, then 15–25 second steeps. 7–10 steeps. {% elif category.id == 'black' %} Use 5g per 100ml, water at 90–95°C. Steep for 15–20 seconds, increasing. 5–6 steeps. {% elif category.id == 'dark' %} Use 6–7g per 100ml, boiling water. Rinse twice, then 10–15 second steeps. 10–20 steeps. {% elif category.id == 'scented' %} Use 4g per 100ml, water at 80–85°C. Steep for 30 seconds. 4–5 steeps. {% endif %}

Western Style

{% if category.id == 'green' %} Use 2g per 200ml, water at 75–80°C. Steep for 2–3 minutes. 2–3 steeps. {% elif category.id == 'white' %} Use 2–3g per 200ml, water at 80–85°C. Steep for 3–4 minutes. 2–3 steeps. {% elif category.id == 'yellow' %} Use 2g per 200ml, water at 75–80°C. Steep for 2–3 minutes. 2–3 steeps. {% elif category.id == 'oolong' %} Use 3g per 200ml, water at 90°C. Steep for 2–3 minutes. 3–4 steeps. {% elif category.id == 'black' %} Use 2.5g per 200ml, water at 90–95°C. Steep for 3–4 minutes. 2–3 steeps. {% elif category.id == 'dark' %} Use 3g per 200ml, boiling water. Rinse first, steep 3–4 minutes. 4–5 steeps. {% elif category.id == 'scented' %} Use 2g per 200ml, water at 80–85°C. Steep for 2–3 minutes. 2–3 steeps. {% endif %}
